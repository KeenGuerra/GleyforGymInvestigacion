# ALCANCE Y OBJETIVOS

# SistemaGimnasioGleyforGym

## Objetivo General

Desarrollar un sistema web y móvil para automatizar los procesos administrativos y deportivos del gimnasio GLEYFORGYM, permitiendo gestionar clientes, usuarios, membresías, pagos, asistencias, progreso físico, rutinas de ejercicios y planes nutricionales personalizados mediante inteligencia artificial.

---

## Objetivos Específicos

1. Implementar un módulo de gestión de usuarios y control de acceso basado en roles.

2. Implementar un módulo de gestión de clientes y seguimiento de información personal.

3. Implementar un módulo para la administración de membresías y pagos.

4. Implementar un módulo para el registro y control de asistencias.

5. Implementar un módulo de seguimiento del progreso físico.

6. Implementar un módulo de gestión de ejercicios y contenido multimedia.

7. Implementar un módulo de inteligencia artificial para la generación automática de rutinas.

8. Implementar un módulo de inteligencia artificial para la generación automática de planes nutricionales.

9. Implementar una aplicación móvil para clientes.

---

## Alcance del Proyecto

### Incluye

* Gestión de usuarios.
* Gestión de clientes.
* Gestión de entrenadores.
* Gestión de membresías.
* Gestión de pagos.
* Gestión de asistencias.
* Gestión de progreso físico.
* Gestión de ejercicios.
* Gestión de comidas.
* Generación de rutinas mediante IA.
* Generación de nutrición mediante IA.
* Aplicación web.
* Aplicación móvil.
* Gestión de videos mediante Cloudinary.

### No Incluye

* Facturación electrónica SUNAT.
* Pasarelas de pago en línea.
* Integración biométrica.
* Integración con dispositivos IoT.
* Integración con wearables.
* Integración con sistemas ERP externos.
