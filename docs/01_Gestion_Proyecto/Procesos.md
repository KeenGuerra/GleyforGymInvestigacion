# PROCESOS DEL NEGOCIO

# SistemaGimnasioGleyforGym

## Introducción

Los procesos representan actividades específicas que permiten ejecutar los macroprocesos definidos dentro del gimnasio GLEYFORGYM.

---

# P-01 Gestión de Clientes

## Objetivo

Administrar la información personal y deportiva de los clientes.

## Actividades

1. Registrar cliente.
2. Actualizar información.
3. Consultar cliente.
4. Activar cliente.
5. Desactivar cliente.

## Responsable

Administrador.

---

# P-02 Gestión de Membresías

## Objetivo

Administrar los planes ofrecidos por el gimnasio.

## Actividades

1. Registrar membresía.
2. Modificar membresía.
3. Activar membresía.
4. Desactivar membresía.
5. Gestionar beneficios.

## Responsable

Administrador.

---

# P-03 Asignación de Membresías

## Objetivo

Asociar planes de membresía a clientes.

## Actividades

1. Seleccionar cliente.
2. Seleccionar plan.
3. Calcular fecha fin.
4. Registrar precio asignado.
5. Activar membresía.

## Responsable

Administrador.

---

# P-04 Gestión de Pagos

## Objetivo

## Actividades

2. Consultar pagos.
3. Validar estado.
4. Consultar historial.

## Responsable

Administrador.

---

# P-05 Registro de Asistencias

3. Consultar historial.

Administrador / Entrenador.

# P-06 Gestión de Progreso Físico

## Objetivo

## Actividades

2. Registrar porcentaje de grasa.
3. Registrar masa grasa.
4. Registrar masa magra.
5. Registrar medidas corporales.
6. Consultar historial.

## Responsable
