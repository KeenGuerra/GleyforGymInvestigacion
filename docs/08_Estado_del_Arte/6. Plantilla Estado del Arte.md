## Estado del arte que hace el autor(1 página)

El autor presenta un análisis de los trabajos previos relacionados con sistemas de monitoreo de dieta, destacando que existen dos enfoques principales: métodos manuales y métodos automáticos. Los métodos manuales, como diarios alimenticios o aplicaciones donde el usuario registra lo que come, dependen mucho del usuario, lo que genera errores y falta de precisión.
Por otro lado, los métodos automáticos utilizan tecnologías como sensores, dispositivos wearables e inteligencia artificial. Entre estos, destacan los sistemas basados en visión por computadora, que permiten reconocer alimentos mediante imágenes. Sin embargo, muchos de estos sistemas solo estiman calorías y no consideran otros nutrientes importantes.
El autor también menciona varios trabajos que usan redes neuronales convolucionales (CNN) para detectar alimentos, pero muchos presentan limitaciones, como la falta de personalización, baja precisión en comidas complejas o dependencia de bases de datos limitadas.
Además, algunos estudios utilizan sensores o biomarcadores, pero estos suelen ser costosos o poco prácticos para el uso diario. En general, el estado del arte muestra que, aunque hay avances importantes, aún existen limitaciones en precisión, automatización completa y personalización.

## Motivación del autor (críticas del autor a otros trabajos)(0.5 página)

La principal motivación del autor es que los sistemas actuales de monitoreo nutricional tienen muchas limitaciones. Por ejemplo, muchos requieren ingreso manual de datos, lo que resulta incómodo y poco práctico para el usuario.
Además, la mayoría de sistemas solo se enfocan en contar calorías, dejando de lado otros nutrientes importantes como proteínas, grasas, vitaminas o minerales. También se observa que pocos sistemas ofrecen recomendaciones personalizadas según la salud del usuario.
El autor identifica que existe un vacío en sistemas que sean automáticos, precisos y personalizados al mismo tiempo. Por eso propone desarrollar una solución que combine inteligencia artificial, visión por computadora y análisis nutricional completo.

## Descripción del aporte del autor(2 páginas)

El aporte principal del artículo es el desarrollo del sistema NutriVision, el cual permite detectar alimentos automáticamente a partir de imágenes y estimar su valor nutricional de manera precisa.
Este sistema utiliza inteligencia artificial para reconocer los alimentos, calcular su cantidad y proporcionar información nutricional detallada, incluyendo calorías, proteínas, carbohidratos y otros nutrientes.
Uno de los aportes más importantes es la personalización, ya que el sistema toma en cuenta datos del usuario como su historial médico, preferencias alimenticias y su índice de masa corporal (IMC). Con esto, genera recomendaciones adaptadas a cada persona.
Además, NutriVision incorpora un chatbot inteligente que interactúa con el usuario, responde preguntas y ofrece recomendaciones en tiempo real. También incluye alertas cuando el usuario consume alimentos con exceso de azúcar o carbohidratos.
Otro aporte importante es la integración con sistemas de salud inteligente (Smart Healthcare), donde se conectan diferentes dispositivos para monitorear la salud del usuario.
En general, el sistema busca mejorar los hábitos alimenticios y prevenir enfermedades mediante una solución automatizada, precisa y personalizada.

## Proceso para obtener el aporte que considera el autor(1. página)

El autor utiliza principalmente técnicas de inteligencia artificial y visión por computadora. En específico, emplea el modelo Faster R-CNN para detectar alimentos en imágenes.
El proceso comienza con la captura de una imagen del alimento usando un smartphone. Luego, el sistema detecta los objetos (comida) en la imagen y calcula su tamaño utilizando un objeto de referencia (una moneda).
Después, se realiza el cálculo del volumen del alimento y se convierte en peso. Con estos datos, se consulta una base de datos nutricional para obtener los valores correspondientes.
Finalmente, el sistema aplica algoritmos de recomendación (como filtrado colaborativo y basado en contenido) para generar sugerencias personalizadas. También se utiliza procesamiento de lenguaje natural (NLP) para analizar información del usuario.

## Proceso para resolver el problema considerado por el autor(1. página)

El autor realiza pruebas utilizando un conjunto de imágenes para evaluar el rendimiento del sistema. Se utilizaron aproximadamente 200 imágenes para validar la detección y clasificación de alimentos.
Los resultados muestran que el sistema tiene alta precisión en la clasificación de alimentos, aunque presenta dificultades cuando hay varios alimentos muy juntos en una misma imagen.
El sistema sigue un flujo de trabajo que incluye:
Captura de imagen
Detección del alimento
Cuantificación
Estimación nutricional
Análisis y recomendaciones
Las pruebas demuestran que el sistema puede estimar correctamente los nutrientes y ofrecer recomendaciones útiles para el usuario.

## Observaciones y/o críticas suyas al artículo(0.5 página)

El artículo presenta una propuesta bastante completa e innovadora, pero tiene algunas limitaciones. Por ejemplo, el sistema depende de una imagen clara del alimento, lo que puede afectar la precisión en situaciones reales.
Además, el uso de un objeto de referencia (moneda) puede ser poco práctico para los usuarios. También el dataset utilizado es relativamente pequeño, lo que podría afectar el rendimiento en diferentes tipos de comida.
Otra observación es que no se evalúa el sistema en un entorno real con muchos usuarios, lo que sería importante para validar su efectividad.
A pesar de esto, el trabajo es muy interesante porque combina varias tecnologías (IA, visión computacional y salud) y aporta una solución bastante completa al problema del monitoreo nutricional.
